# Example Micro-SaaS Apps

This directory contains example apps demonstrating what the MicroSaaS Apps platform can generate. Each app is a standalone React component following the platform's constraints:

## Constraints

- ✅ Pure JavaScript (no TypeScript)
- ✅ No import/export statements
- ✅ Tailwind CSS for styling
- ✅ Self-contained and functional
- ✅ Works with Babel JSX transformer in browser
- ✅ Client-side only (no server dependencies)

## Example Apps

### 1. Color Palette Generator
**Price:** $0.50 | **Category:** Creative

Generates random 5-color palettes with click-to-copy functionality. Beautiful gradient backgrounds and hover effects.

### 2. Tip Calculator
**Price:** $0.75 | **Category:** Finance

Calculate tips with preset percentages, custom slider, and bill splitting. Shows tip amount, total, and per-person cost.

### 3. Pomodoro Timer
**Price:** $1.00 | **Category:** Productivity

Full-featured pomodoro timer with work/break modes, visual indicators, and completed session tracking. Auto-switches between work and break periods.

## Testing These Apps

These apps can be:
1. Copied into the app builder for testing
2. Used as templates for AI generation prompts
3. Deployed to the BSV blockchain via the platform
4. Previewed in the test environment

Each app demonstrates different UI patterns and interactive features that the AI can generate.
