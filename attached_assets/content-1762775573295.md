### Desktop Downloads

Grab the latest signed installer for macOS, Linux, or Windows and verify the optional checksums as needed.

Install the native desktop build tailored for your Mac hardware.

- [Download for Apple Silicon\\
DMG installer (arm64)](https://desktop-binaries.getmetanet.com/metanet-client-macos-arm64.dmg)
- [Download for Intel Macs\\
DMG installer (x64)](https://desktop-binaries.getmetanet.com/metanet-client-macos-x64.dmg)

Grab the portable AppImage and verify with published signatures and hashes.

- [Download AppImage\\
Desktop AppImage (x86\_64)](https://desktop-binaries.getmetanet.com/metanet-client-linux-x86_64.AppImage)
- [GPG Signature](https://desktop-binaries.getmetanet.com/metanet-client-linux-x86_64.AppImage.asc)
- [SHA256 Checksums](https://desktop-binaries.getmetanet.com/SHA256SUMS)
- [Checksums Signature](https://desktop-binaries.getmetanet.com/SHA256SUMS.asc)

Verify with `sha256sum --check SHA256SUMS` and `gpg --verify SHA256SUMS.asc`.

Install the signed Windows builds for the best desktop experience.

- [Download MSI Installer\\
Digitally signed MSI (x64)](https://desktop-binaries.getmetanet.com/metanet-client-windows-x64.msi)
- [Download EXE Installer\\
Signed NSIS installer (x64)](https://desktop-binaries.getmetanet.com/metanet-client-windows-x64.exe)

Builds are signed with DigiCert Software Trust Manager certificates.
